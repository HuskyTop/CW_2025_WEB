Технічна реалізація симуляції повітряного потоку

Симуляція моделює поведінку повітряного потоку, який обтікає профіль крила або іншу перешкоду. Вона заснована на базових принципах аеродинаміки, де враховуються такі параметри як густина повітря, швидкість потоку, рівень турбулентності, довжина та товщина перешкоди.

Веб-додаток побудований на основі JavaScript та Canvas API, що дозволяє створювати інтерактивну графіку прямо в браузері. Користувач задає початкові параметри, які потім обробляються та використовуються для побудови візуальної симуляції потоку частинок.

Для збереження та завантаження симуляцій використовується локальна база даних SQLite, взаємодія з якою відбувається через сервер на Node.js. Це дозволяє кожному користувачу мати доступ до своїх збережених результатів і швидко повторювати експерименти.

На зображенні нижче зображено умовну схему профілю крила, який використовується для візуального представлення взаємодії з потоком:

Схема профілю крила для моделювання потоку

Під час симуляції частинки рухаються відповідно до визначених параметрів, імітуючи ламінарний або турбулентний потік залежно від умов. Завдяки візуалізації користувач може наочно побачити вплив зміни параметрів на поведінку повітря.
